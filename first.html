//Static: The position: static; is the default position value. Elements with position: static; are positioned according to the normal document flow.//
Static
//Relative: The position: relative; positions an element relative to its normal position in the document flow.//
Relative
//Absolute: The position: absolute; removes an element from the normal document flow and positions it relative to its closest positioned ancestor. //
Absolute
//Fixed: The position: fixed; positions an element relative to the browser window, and it remains fixed in its position even when the page is scrolled.//
Fixed
//Sticky: The position: sticky; positions an element based on the user's scroll position. It behaves as position: relative; until a specific threshold is reached, then becomes position: fixed;//
Sticky
